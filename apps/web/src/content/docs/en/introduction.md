CodeSnip is a cross-platform, lightweight desktop sticky-note app. Its main twist is **better support for pasted code**:

- Watch clipboard text, guess language, and highlight with Shiki
- Translucent, always-on-top, frameless sticky windows
- Zoom, opacity, and click-through controls

If you often show code in screenshots, recordings, or live explanations, it is more direct than pasting into an image editor first.
